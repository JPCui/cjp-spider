    类别（class）（id,name类名）
    站点（site）（id,name站点名,domain域名）
	属性（attributes）（id,classId类别ID,name属性名,value）
	*规则类别（rules_type）（id,type[html/json]）（分别对应不同的抓取方法）
	属性抓取规则（rules）（id,attrId属性ID,ruleId规则ID,rule规则）

例---------------------------

	属性：标题、时间、内容、作者。。。
	类别：博客、新闻。。。
	站点：新浪博客，CSDN博客。。。
	规则：标题抓取规则、时间抓取规则、内容抓取规则。。。
	规则类别：JSON、HTML、XML
